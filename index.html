//Weather Map Application

This application focuses less on the aesthetics and more on the functionality. Highlighting my skills with using jQuery AJAX requests, JavaScript and using both the Google Maps API and Open Weather Map API. Each request made by jQuery is first send to the Google Maps API to retrieve the geocode for the nearest city to either the searched city or clicked location. Google Maps interprets and returns the location. A new AJAX request is sent to the Open Weather Map API using latitude and longitude. The request returns a JSON object which is parsed and subsequently rendered to the page.
